Using the browser Gamepad API with WAMP and Crossbar.io

This demo shows how to use the new W3C Gamepad API to connect your gamepad to the cloud and devices via real-time messaging using WAMP (which runs over WebSocket) and Crossbar.io.

This demo works best with Firefox, also Chrome, but no other browser currently (2014/11). To test support and basic operation of your gamepad (without any real-time messaging), you can checkout this demo.

Tested with Microsoft Xbox 360 wireless controller and PC wireless receiver. See here.

To test, open the Gamepad Monitor in a first browser (on any PC).

Then, on a second PC (where you gamepad is connected), open the Gamepad Connector.

You should see button presses and joystick movements being received in real-time in the first browser (and any other instances connected).
